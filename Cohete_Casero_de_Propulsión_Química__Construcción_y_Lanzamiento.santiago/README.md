Cohete casero — Aceleración por Propulsión Química

Proyecto: informe en LaTeX que describe la construcción y pruebas de un cohete casero propulsado con vinagre y bicarbonato.

Archivos:
- `main.tex` — documento principal en LaTeX.

Cómo compilar (local):
1. Instala una distribución TeX (TeX Live o MikTeX).
2. En la carpeta, ejecuta `pdflatex main.tex` (dos veces para referencias).

Notas:
- Añade la imagen `image.jpg` si quieres que la figura se muestre.
- Revisa y corrige rutas si mueves archivos dentro del repo.
